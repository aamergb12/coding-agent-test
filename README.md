# Python Calculator

A modular calculator implementation with separate operation files.

## Structure
- `main.py`: Entry point of the application
- `calculator.py`: Main Calculator class implementation
- `operations.py`: Basic mathematical operations
- `utils.py`: Utility functions for input validation
- `constants.py`: Constants and configuration

## Usage
Run the calculator using:
```bash
python main.py
```

## Features
- Basic arithmetic operations (+, -, *, /)
- Input validation
- Error handling
- Modular design
